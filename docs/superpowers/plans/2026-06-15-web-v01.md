# Web v0.1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a local read-only FastAPI/Jinja strategy dashboard backed by `stock_history.db` and `stock_signals.db`.

**Architecture:** Keep Web code isolated under `web_app/`. Service modules read SQLite and return plain dictionaries/lists. FastAPI routes render Jinja templates; no route mutates strategy data or performs trading actions.

**Tech Stack:** FastAPI, Jinja2, Uvicorn, SQLite, unittest.
